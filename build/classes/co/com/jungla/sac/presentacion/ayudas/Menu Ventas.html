SACJUNGLA - SISTEMA ADMINISTRATIVO PARA LA EMPRESA DE BORDADOS INDUSTRIALES "JUNGLA"

VENTAS

En este menu se encuentra los siguientes items que permiten la realización de este proceso:

Registros

  • Nuevo Tipo Cliente: registra toda la información de los distintos tipos de cliente que pueden existir. Esta utilidad es necesaria para registrar un Nuevo Cliente
  • Nuevo Cliente: registra la información del cliente incluyendo su tipo, siendo esta necesaria para el registro de la Venta de Articulos.
  • Nuevo Forma de Pago (Cliente): registra la Forma de Pago que utilizará el Cliente, ya sea de Contado o Crédito.
  • Nuevo Banco: permite agregar todos los datos necesarios que requiere el banco, este registro es muy necesario para los medios de pago que el cliente utilizará para el pago de la venta de contado o un abono a una venta a crédito.
  • Nuevo Vendedor:: permite eliminar la Unidad de Medida.

Venta de Articulos

Esta utilidad permite la venta de los artículos generando un Recibo de Caja en caso de pago inmediato y el caso de quedar pendiente la venta toma el estado pendiente de pago. En ella puede establecer el Cliente, el Vendedor, la forma de pago y los articulos en el Inventario..

Caja / Recaudos

Permite registrar los pagos totales y los abonos de las facturas de venta. Permite consultar de manera ágil todas las ventas pendientes de pago y recibirlos con diferentes medios de pago. Ademas ofrece la posibilidad mediante un botón la creación de Anticipos a Cliente.

Devoluciones de Clientes (Nota Crédito)

Esta utilidad le permite registrar Devoluciones por concepto de garantías y/o errores en la Venta a los Clientes. Cada devolución genera una Nota Crédito que puede ser utilizada como medio de pago por parte del Cliente.

Cotizaciones

Puede generar Cotizaciones para sus clientes existentes o para clientes nuevos que no esten en la base de datos. Las cotizaciones se consultar para ser cargados como factura de venta.

Consultas

  • Bancos: permite consultar la información de los Bancos. Información que puede ser modificada.
  • Clientes: permite consultar la información de los Clientes. Información que puede ser modificada.
  • Cotizaciones: permite consultar la información de las Cotizaciones. Información que puede ser modificada.

Reportes

  • Reporte de Ventas: este reporte le permite consultar todas las Ventas realizadas.
  • Reporte de Cartera Pendiente: reporte que permite consultar todas las Ventas Pendientes de Pago.
  • Reporte de Recibos de Caja: este Reporte le permite consultar todos los Recibos de Caja que se generaron en un periodo de tiempo por conceptos de Ventas Pendientes.
  • Reporte de Notas Crédito:: es un reporte de todas las Notas generadas por Devoluciones de sus Clientes en un periodo de tiempo seleccionado.
