SACJUNGLA - SISTEMA ADMINISTRATIVO PARA LA EMPRESA DE BORDADOS INDUSTRIALES "JUNGLA"

BIENVENIDO

Software Administrativo diseñado para La Empresa de Bordados Industriales "Jungla". El programa permite llevar una gestion de las COMPRAS, INVENTARIOS Y VENTAS, procesos generales que principalmente maneja la compañia diariamente. Además, cuenta con todos los reportes y consultas necesarias para que el usuario pueda desarrollar buenas decisiones.

La aplicación esta desarrollada como Aplicación de Escritorio, constituida por un servidor de Bases de Datos (MySQL), una plataforma JAVA JEE y un servidor GLASSFISH. Este software fue desarrollado con interfaz facil de manejar.

El principal objetivo del programa es que quien lo maneja día a día pueda encontrar todas las herramientas necesarias para controlar facilmente el área de Ventas, Inventarios, y Compras. No se necesita tener conocimiento contables para su manejo ya que esta diseñado para que cualquier usuario independientemente de sus conocimimentos lo manipule facilmente, pues el programa puede hacer lo siguiente:

PRINCIPALES CARACTERISTICAS

  • Ambiente Windows
  • Generacion e impresion de reportes y documentos
  • Catalogo de Productos con Foto
  • Manejo de Compras y Ventas
  • Control de Egresos y Cuentas x Pagar
  • Control de las Devoluciones tanto para las Compras como para las Ventas
  • Control de la Cartera Pendiente y Recaudos
  • Manejo de Kardex Electronico
  • Control de Inventarios
  • Entrada y Salidas de Inventario
  • Cuadres de Caja Diarios
  • Sistema de Caja con múltiples Medios de Pago
  • Generación de Cotizaciones y Ordenes de Compra
  • Exportación de Hojas de Calculo
  • Generacion del Balance General y Estado de Resultados (GyP)
  • Control de Anticipos tanto para Proveedores como para Clientes
